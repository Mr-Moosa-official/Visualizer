# **App Name**: Visualizer

## Core Features:

- Limit Configuration: Set request limits (requests per minute, hour, day).
- Request Simulation: Simulate sending API requests to test the rate limiter.
- Real-time Monitoring: Display real-time counts of allowed and blocked requests.
- Visual Meter: Graphical meter to show current rate limit usage.

## Style Guidelines:

- Primary color: A vibrant green (#90EE90) for 'allowed' indicators.
- Background color: Dark gray (#282828) provides a neutral, sophisticated backdrop for the visualizer.
- Accent color: A strong red (#FF4500) to clearly indicate 'blocked' requests, contrasting sharply with the dark background and primary color.
- Body and headline font: 'Inter', a sans-serif font for a modern, clear, and easily readable interface.
- Simple, clear icons for configuration and status indicators.
- Dashboard layout with clearly defined sections for configuration, simulation, and monitoring.
- Subtle animations to highlight changes in request status and meter levels.
